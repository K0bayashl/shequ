# 社区公约（简版）

> 适用于当前 CDK 注册制的小圈子程序员社区。

## 1. 基本原则

- 社区以学习交流为主，鼓励高质量分享。
- 尊重他人，禁止人身攻击、引战和恶意骚扰。
- 禁止发布恶意代码、钓鱼链接、违法违规内容。
- 不刷屏、不灌水、不重复发广告。

## 2. 内容发布规则

- 会员发布内容默认无需预审。
- 平台会保留最小安全防护，对明显高风险内容可自动拦截或隐藏。
- 如发现问题内容，任何成员都可举报。

## 3. 违规处理规则

- 轻度违规：提醒或警告，必要时删除内容。
- 中度违规：删除内容并短期限制发布。
- 重度违规：立即下架内容并封禁账号。

## 4. 处理时效

- 高风险问题：24 小时内处理。
- 一般问题：72 小时内处理。

## 5. 申诉与复核

- 被处理用户可提交一次申诉说明。
- 管理员会根据记录复核并给出最终结果。

## 6. 说明

- 本社区为单人维护，采取发布后处置策略，不做全量人工预审。
- 规则会根据实际运行情况持续更新。
