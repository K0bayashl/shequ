## 14. 给 AI 的输出要求

如果你希望 Claude 或 Copilot 按固定格式输出分析结果，可以在需求文档末尾补上这一段。

- 请先给出需求理解。
- 请列出当前假设。
- 请判断任务范围。
- 请拆分实现步骤。
- 请指出风险与待确认项。
- 请给出推荐的实现顺序。
- 如果信息不足，请先提最少必要问题，不要直接猜。

## 15. 简短示例

### 示例：新增用户导出功能

- 需求名称：用户导出
- 背景说明：运营需要导出符合条件的用户列表进行分析。
- 需求目标：支持按筛选条件导出 Excel。
- 非目标：暂不支持 PDF 导出。
- 功能需求：
  - 支持按注册时间筛选
  - 支持按用户状态筛选
  - 导出文件包含用户 ID、姓名、手机号、注册时间
- 验收标准：
  - 当筛选条件有效时，能生成正确 Excel 文件
  - 当没有符合条件的数据时，应返回空文件或明确提示
  - 当导出失败时，应有清晰错误提示
- 风险与待确认项：
  - 导出数据量过大时是否需要异步处理
  - 手机号是否需要脱敏
